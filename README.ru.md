# Full Auto Bot — Brotato

[English version →](README.md)

![preview](preview.png)

Мод для Steam Workshop, добавляющий **кнопку с роботом** на экран выбора
сложности Brotato. Нажми один раз — и бот целиком проходит за тебя забег
на **Danger 6 (Nightmare)**: движение, магазин, прокачка уровней,
сундуки — с 1-й волны до 21-й и дальше в Endless.

Поддерживает все **64 персонажа** (50 базовых + 14 из DLC *Abyssal
Terrors*).

---

## Что делает

- Добавляет дополнительную опцию на экране сложности рядом с **D0–D6**.
- Автоматически выбирает **Danger 6** при нажатии кнопки бота.
- Индивидуальные стратегии для каждого из 64 персонажей: своя
  идентичность билда, семейства оружий, запрещённые предметы, стиль
  движения.
- Дальнобойные кайтеры, ближнебойные танки, безоружные билды (Beast
  Master, Bull), пацифистский стек ладоней, билды на взрыв при уроне —
  у каждого персонажа свой стиль игры.
- Настоящая оценка по **DPS / эффективному HP** при выборе предметов в
  магазине и карточек уровня. Никакой наивной эвристики по статам.
- Множественные стратегии движения: sampling-flee, орбитальное
  отступление, чистое отталкивание, аварийный panic-dodge, динамическая
  осторожность к снарядам.
- Защита от «зеркальной ловушки пуль» — перпендикулярный escape, когда
  два снаряда взаимно гасят силу отталкивания.

## Как пользоваться

1. Подпишись на мод в Steam Workshop *(или скачай zip последнего релиза
   из этого репозитория и положи в папку `mods/` игры).*
2. Убедись, что **ModLoader** установлен и включён.
3. Запусти Brotato. В главном меню активируй **Full Auto Bot** в разделе
   *Mods*.
4. Выбери персонажа и стартовое оружие как обычно.
5. На экране сложности появится новая иконка с роботом. Нажми её.
6. Бот возьмёт управление и пройдёт забег на **Danger 6**.

В любой момент можно вернуть управление — нажми клавишу движения, и
бот мгновенно уступит ручному вводу.

## Совместимость

- Brotato **1.1.15+**
- Требует **ModLoader 6.x**
- Работает с персонажами DLC *Abyssal Terrors*
- Только одиночная игра
- Linux (Steam Proton), Windows, macOS — чистый GDScript, без нативных
  бинарей

## Известные ограничения

- Первые волны забега зависят от RNG. Бот не перезапускает забег при
  неудачном старте.
- Часть стратегий для DLC-персонажей основана на мете сообщества — они
  будут улучшаться по мере получения обратной связи.
- Бот не переходит в Endless автоматически после босса 20-й волны —
  играет в том режиме, который ты выбрал на старте.
